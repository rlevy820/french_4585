Une étiquette d'object

Titre

Autoportrait à travers les Citations

Auteur

Ryan Levy

Identification de l'objet

Site web interactif

Les matériaux/médias employés

HTML, CSS, JavaScript

Description

Un site web interactif qui présente un autoportrait à travers une collection de citations significatives. Chaque citation est présentée sur une carte interactive qui révèle une réflexion personnelle lorsqu'elle est cliquée. Le site combine des éléments visuels modernes avec une narration personnelle pour créer une expérience immersive.
